course reflection : where i used advanced css

Throughout my refactored portfolio, I pulled different techniques from Jonas Schmedtmann’s Advanced CSS and Sass course and applied them to the pages and their elements. I followed his “think, build, architect” mindset by planning the layout for each page with clean SASS structure, BEM, and accessibility in mind.

Natours cards

I recreated the Natours card component with a 3D flip hover using perspective, backface-visibility, and rotateY().

Responsiveness

I built the site mobile-first, then added tablet + desktop breakpoints at 600px and 1024px.

Hover transitions

I applied smooth hover transitions on interactive elements like preview stacks, links, and buttons.

Animations

I used keyframes for controlled motion like the ticker and subtle snap-like motion on visuals.

Reveal effects

On the resume page, IntersectionObserver triggers slide-in reveals as the user scrolls.

Blur-in reveal

I added a blur + fade reveal for the resume preview image so it appears smoothly.

Form validation

The contact page includes validation states (errors + success message).

Reduced motion

I added prefers-reduced-motion support to disable key animations and scroll reveals.
